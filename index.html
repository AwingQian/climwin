climwin

climwin is a software package for detecting and analysing climate windows in the R statistical computing environment

To get started read our published work in Methods in Ecology and Evolution and PLoS ONE.

Our package documentation includes both an introductory and an advanced vignette that provide an overview of important features included with the package. A fully inclusive user manual with FAQs is currently a top priority and will replace these vignettes.

climwin is a versatile toolbox for climate window analyses in the R statistical computing environment. Climate window analysis is an exploratory method in which users can test for periods of climatic sensitivity, where variation in climate strongly drives variation in a biological trait. climwin allows users to test the impact of all possible climate windows within a specified range, comparing windows using an information criterion approach. This approach removes the need to designate arbitrary windows over which we measure climate, which often occurs with little biological justification.

Main features of the package:

Statistical analysis

  • Sliding window analysis.
  • Weighted window analysis (Weibull and Generalised Extreme Value distributions).
  • k-fold cross validation in window selection.
  • Methods to guard against false positive results (comparison to randomised data).
  • Range of summary statistics for testing climate impacts (e.g. mean, max, slope).
  • Ability to test linear, quadratic, cubic, logarithmic and inverse impacts of climate.
  • Compatibility with a range of model functions (e.g. lmer, glmer, coxph).

Visualisation

  • AICc values of all tested climate windows.
  • Model beta values of all tested climate windows (including potential quadratic and cubic terms).
  • Model weight values of all tested climate windows, highlighting the 95% confidence set.
  • Boxplot climate window range within the 95% confidence set.
  • 3D visualisation of AICc and model weights.

Over time we hope to include additional features to improve the versatility of climwin.
